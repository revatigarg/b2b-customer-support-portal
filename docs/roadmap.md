# Roadmap

- Structured support intake
- Ticket categories and routing logic
- Basic knowledge base structure

- Order issue intake taxonomy (shipping, payment, item issues)
- Refund and return workflow mapping
- Inventory discrepancy triage
- Checkout issue troubleshooting paths

## Phase 3. Platform scale
- Agent view dashboard concept
- SLA and escalation visibility
- Integration mapping (CRM or ticketing system)
- Analytics plan for deflection and resolution time
